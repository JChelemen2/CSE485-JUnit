# CSE485-JUnit
Repository for the growing a competency assignment where I learned JUnit testing.

Note: I originally used a java 21 distribution from Amazon since it was the default selected in IntelliJ, but it is missing some of the required functions such as BigDecimal and I couldn't get Maven to index properly to search it within the locator (if it is there at all, the website doesn't have a comprehensive package). So I recreated the project with a standard Oracle 21 JDK from their website and it had the functions included so they could be imported. If it doesn't work, try swapping the java environment.
